🧠 Interview Questions
🧠 Q1: How do you structure an ML project?
👉 Answer:
“I structure ML projects using modular architecture with separate layers for data ingestion, feature engineering, modeling, pipelines, and configuration, ensuring scalability and maintainability.”

🧠 Q2: Why use config files?
👉 Answer:
“To separate code from parameters, enabling reproducibility and easy experimentation.”

🧠 Q3: Why not use notebooks for production?
👉 Answer:
“They lack modularity, version control, and scalability.”
